## LinkedIn Version

```text
For the Build Small Hackathon, I built AI Prophecy Court: a Gradio app where
public AI predictions become source-linked court cases.

The core interaction is intentionally AI-native. Two sub-32B models receive the
same preserved evidence packet and produce competing roasts. The visitor votes
before the model identities are revealed, then can generate a source-stamped
verdict card.

The stack is Gradio Server, React, TypeScript, json-render, a typed Python
backend, a deterministic docket pipeline, and optional Modal runtime adapters
for live model calls and aggregate feedback.
```
